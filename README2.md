Habit Tracker - יומן הרגלים ומשימות שיתופי
תיאור הפרויקט
Habit Tracker היא אפליקציה מקיפה המשלבת מעקב אחר הרגלים ומשימות עם פלטפורמה קהילתית ייחודית, שמטרתה לשפר את המוטיבציה והאחריות ההדדית. הפרויקט מספק למשתמשים כלים להצבת מטרות אישיות (הרגל/מטרה), תיעוד ההתקדמות היומית (יומן), והצטרפות לקבוצות תמיכה ממוקדות מטרה (קהילה) כמו קבוצת תלמידים או קבוצת עובדים עם יעד משותף.

תכונות מרכזיות
פרופיל משתמש: ניהול חשבונות משתמשים עם אימות מאובטח.

ניהול הרגלים (Habit): הגדרה, מעקב יומי ושבועי אחר רצף ההצלחות והתקדמות במטרות.

רישום יומן (Journal): תיעוד יומי קל ומהיר של הפעולות שבוצעו, המקושר להרגלים הרלוונטיים.

קהילות ומטרות שיתופיות: יצירת קבוצות (Community) סגורות או פתוחות, המאפשרות לחברים לעקוב יחד אחר מטרה משותפת ולעודד אחד את השני.

ממשק API: ממשק RESTful (או GraphQL) יציב ומתועד לתקשורת מול צד הלקוח.

מבנה נתונים וארכיטקטורה
הפרויקט מתבסס על ארכיטקטורה הכוללת API ו-4 מודלים מרכזיים במסד הנתונים:

טבלה / מודל	תיאור	יחסים מרכזיים
User	פרטי המשתמש, אימות (Auth) והרשאות.	יחיד לרבים עם Journal, Habit, ו-Community (כיוצר).
Habit / Goal	הגדרת ההרגל/המטרה, תדירות נדרשת, וסטטוס מעקב.	יחיד לרבים עם User ו-Journal.
Journal	רשומת רישום יומית של פעולות והתקדמות.	יחיד לרבים עם User.
Community	קבוצה שיתופית עם מטרה משותפת.	רבים לרבים עם User (חברי קהילה).

התקנה והרצה
שיבוט המאגר:

Bash

git clone [הוסף כתובת ה-Git שלך כאן]
cd habit-tracker
הגדרת משתני סביבה:

צור קובץ בשם .env בתיקייה הראשית והגדר את משתני הסביבה הנחוצים:

PORT=3000
DATABASE_URL="[כתובת החיבור למסד הנתונים]"
JWT_SECRET="[מפתח סוד חזק לאימות]"
# הוסף משתנים נוספים כנדרש...
התקנת תלויות וטיפול במסד הנתונים:

Bash

# התקנת כל התלויות הנדרשות
npm install

# הפעלת סקריפט ליצירת סכמה/טבלאות במסד הנתונים
npm run migrate
הפעלת השרת:

Bash

npm run start
השרת אמור לפעול כעת בכתובת: http://localhost:3000

תיעוד API
ממשק ה-API זמין בכתובת הבסיס http://localhost:3000/api/v1.

[המלצה: כדאי ליצור קובץ תיעוד API מלא (לדוגמה, באמצעות OpenAPI/Swagger) ולספק קישור אליו כאן.]

נקודות קצה (Endpoints) לדוגמה
מתודה	נתיב	תיאור
POST	/api/v1/users/register	יצירת משתמש חדש.
POST	/api/v1/habits	יצירת הרגל/מטרה חדשה. (דורש אימות)
POST	/api/v1/journals	הוספת רשומת יומן יומית. (דורש אימות)
GET	/api/v1/communities/join/:id	הצטרפות לקהילה קיימת. (דורש אימות)
