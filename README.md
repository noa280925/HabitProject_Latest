#  HabitConnectAPI  
### מערכת לניהול הרגלים אישיים וקהילות שיתופיות  
פרויקט ASP.NET Core Web API שנבנה במסגרת קורס פיתוח ב-‎.NET.  
המערכת מאפשרת למשתמשים לעקוב אחרי הרגלים, לתעד הצלחות ביומן אישי, ולהצטרף לקהילות עם מטרות משותפות.

---

##  :תכולת הפרויקט
- ניהול **משתמשים (Users)**  
- ניהול **הרגלים (Habits)**  
- ניהול **רישומי יומן (Journals)** — רישום יומי של הצלחות/כישלונות בהרגלים  
- ניהול **קהילות (Communities)** — קבוצות של משתמשים עם מטרה משותפת  
- חישוב **מדד השפעה (Impact Score)** לקהילה — כמה אחוז מההרגלים הושלמו בהצלחה  
- **Swagger UI** מובנה לבדיקות והרצה  
- בסיס נתונים In-Memory (ללא התקנה חיצונית)  
- מוכן להרחבה לשימוש ב־SQL Server / SQLite / ADO.NET

---

## :טכנולוגיות בשימוש
| רכיב | תפקיד |
|-------|--------|
| **ASP.NET Core Web API (.NET 8)** | לבניית ממשק שרת RESTful |
| **C#** | שפת התכנות הראשית |
| **Entity Framework Core (InMemory)** | גישה לנתונים בזמן הפיתוח |
| **Swagger (Swashbuckle)** | ממשק אינטראקטיבי לבדיקה |
| **xUnit** | סביבת בדיקות יחידה (Unit Tests) |
